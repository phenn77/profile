Information

Split Bill is a system to help users on splitting bills they have. All the users need to do are to insert the participants' name and the invoices, choose the splitting to individually or evenly to all participants and the results will come out.

Application is made with :

Pictures

Login Page

Description

Login page for users. It is mandatory because all the data will be stored in database.

Index Page

Description

Index page to show all the list of transactions made.

View Transaction

Description

View of Transaction document with all the lists of invoice inside and participants.

Create Invoice

Description

Invoice Form to input all the data. There are 2 choices for the dividing amount, Split Individually and Split Evenly.

View Invoice

Description

The view of an Invoice after it is inputted, with the information of each participant's orders and amount needed to pay.

List Amount

Description

In Transaction view, each participants will be showed with list of invoices each of them participated.

For payment, it will be in manual, the view works only as information for the participants to inform them whether they have paid or not.
